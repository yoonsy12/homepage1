# 🏫 6학년 우리반 홈페이지

우리반 친구들을 위한 학급 홈페이지 소스코드입니다.

## 👋 소개
이 홈페이지는 초등학교 6학년 학급을 위해 만들어졌습니다.
다음과 같은 기능이 포함되어 있습니다:

- **홈 화면**: 환영 메시지와 학교 이미지
- **가정통신문**: 알림장 및 공지사항 확인
- **로그인/회원가입**: 이메일 인증을 통한 실제 회원가입 및 로그인
- **실시간 연동**: 선생님이 글을 쓰면 학생들 화면에 바로 뜨는 기능 (Firebase)
- **관리자 모드**: 선생님 계정으로 로그인했을 때만 글쓰기 가능

## 🚀 사용 방법

### 1. 내 컴퓨터에서 실행하기
`index.html` 파일을 다운로드하여 크롬(Chrome)이나 엣지(Edge) 브라우저에서 열면 바로 실행됩니다.

### 2. 깃허브 페이지(GitHub Pages)로 웹사이트 만들기
1. 이 저장소(Repository)를 자신의 깃허브 계정으로 포크(Fork)하거나 다운로드 후 업로드합니다.
2. 파일 이름이 `홈페이지 만들기.html`이라면 **`index.html`**로 이름을 변경해주세요.
3. 저장소의 **Settings(설정) > Pages** 메뉴로 이동합니다.
4. **Branch**를 `main` (또는 `master`)으로 설정하고 저장합니다.
5. 잠시 후 생성된 주소로 접속하면 홈페이지가 인터넷에 공개됩니다.

## 🛠️ 기술 스택
- HTML5
- CSS3
- JavaScript (Vanilla)
- Firebase (Authentication, Firestore)

## ⚠️ 설정 안내
이 홈페이지는 **Google Firebase**와 연결되어 있습니다.
정상적으로 작동하려면 Firebase 콘솔에서 **Authentication(이메일 로그인 사용 설정)**과 **Firestore Database(데이터베이스 생성)**가 활성화되어 있어야 합니다.

### 데이터베이스 규칙(Rules) 설정
데이터가 저장되지 않는다면 Firestore Rules를 다음과 같이 변경해보세요 (테스트용):
```
allow read, write: if true;
```
